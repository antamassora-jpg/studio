================================================================================
                        TROUBLESHOOTING
================================================================================

IF DATA NOT SYNCING:
  1. Check browser console (F12 → Console)
     - Look for red error messages
     - Copy error and search in SUPABASE_MIGRATION_GUIDE.md
  
  2. Check Network tab (F12 → Network)
     - Add new data
     - Look for POST request to /api/students
     - Check response status (should be 200)
     - Check response body (should contain new student data)
  
  3. Check Supabase Dashboard
     - Login to https://app.supabase.com
     - Select project "studio"
     - Table "students" → see if data is there
     - If not there, API error
     - If there but not showing in app, client-side bug

COMMON ISSUES & FIXES:
  • Error "Failed to fetch"     → API route down, restart server
  • RLS Policy Error            → Disable RLS for testing in Supabase dashboard
  • Data shows in DB but not UI → Check refetch() is called after update
  • Infinite loading            → Check loading state handling in component
